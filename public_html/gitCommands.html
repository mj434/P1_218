Git Commands
With Examples

There are various Git commands used repeatedly throughout the course.

  1. git init

    2. This command is used to create a git repository.

  2. git add

    3. This command is used to obtain a summary of which files have changes that are staged for the next commit.

  3. git status

    4. This command displays the current state of the working directory.

  4. git commit -m ""

    5. Commit is another word for Snapshot. The git commit command is used for taking a screenshot of the current directory.

    Notice that it needs a message between the quotations

  5. git checkout (-b)

    6. This command creates a new branch and jumps there.

  6. git push

    7. This command is used to upload your local repository content to a remote repository

  7. git pull

    8. This command is used to download content from a remote repository into a local repository, while updating the local repository to match that content.
