Apache Web Server Commands

An Apache Server is a web server application that delivers web server content over the internet,suck as HTML, CSS, and more. Apache is a community-developed web application published by the Apache Software Foundation.

  1. $ sudo apt install apache2

    2. This command is used to install Apache server [on Debian/Ubuntu].

  2. $ sudo httpd -v OR $ sudo apache2 -v

    3. These commands are used for checking the installed version of your Apache server

  3. -V

    4. This command is used to display the Apache version number and compile settings

  4. $ sudo httpd -t OR $ sudo apache2ctl -t

    5. These commands are used for checking the Apache configuration files for any syntax errors.

  5. $ sudo systemctl start apache2 [On Systemd] // $ sudo service apache2 start [On SysVInit]

    6. These commands are used to start the Apache server [on Ubunt, Debian]

  6. $ sudo systemctl enable apache2 [On Systemd] // $ sudo chkconfig apache2 on [On SysVInit]

    7. These commands enable the Apache server to auto-start at system boot. [on Ubunt, Debian]

  7. $ sudo systemctl stop apache2 [On Systemd] // $ sudo service apache2 stop [On SysVInit]

    8. These commands stop the Apache service. [on Ubunt, Debian]
