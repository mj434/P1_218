This is a website for project 1 made my Mohit Joshi and Omar Elserty.
Below is the step by step walkthrough of how the project was accomplished:
  1. Create an empty repository on GitHub with appropriate contributors.
    2. First, as we need a apace where all members of the team can contribute remotely, we set up a GitHub repository and add required contributors.
    Then, we clone the repository into our own machines so we can start working.
  2. Build the actual webpages.
    3. We then proceed to build the webpages as assigned under GitHub issues.
  3. Push your work onto GitHub.
    4. We push our work onto GitHub using branches that merge into our main code using pull requests.
    As we merge pull requests, we link our issues with them so that they automatically close.
  4. Create a Docker Container
    5. Using Dockerfile and docker-compose.yml files, we create a docer container to host our sites on our local machine.
  5. Push the container image to DockerHub Repository.
    • Build the container image.
    • Tag your existing local image to the image on DockerHub.
    • Commit your changes.
    • push your image to the repository.
  6. Log into azure through command line and create an aci context.
    7. Use "docker login azure" to log in to azure.
    Use "docker context create aci [name]" to create an aci context.
    Switch to your aci context using "docker context use [name]".
  7. Deploy your container on Microsoft Azure.
    8. Use the command in slack "docker run --domainname project -dp 80:80 mj434/p1_218" to deploy the container on azure.
