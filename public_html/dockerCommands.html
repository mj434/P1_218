Docker Commands

Docker is an open source containerization platform. It enables developers to package applications into containers.

  1. Container

    2. a container is a process on your machine that has been isolated from all other processes on the host machine, which helps in accessing certain features a lot easier than Linux. It can alsobe uploaded to cloud easily.

  2. Container Image

    3. Docker image is a file used to execute code in a Docker container.

  3. Docker File

    4. A Dockerfile is a text document that contains all the commands a user could call on the command line to assemble an image.

  4. Docker Compose

    5. Compose is a tool for defining and running multi-container Docker applications.
